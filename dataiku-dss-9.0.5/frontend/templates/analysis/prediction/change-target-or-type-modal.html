Do you want to keep your settings or have them all re-detected?

Even if you choose the KEEP SETTINGS option, the following settings will be lost:
  • Algorithms — not the same between classification & regression
  • Architecture — the current code will be commented
  • Metrics — not the same between the different prediction types
  • Weighting strategy — the target variable cannot be the sample weight
  • Weighting strategy — regression cannot use class weight
  • Assertions — depend on prediction type & target
