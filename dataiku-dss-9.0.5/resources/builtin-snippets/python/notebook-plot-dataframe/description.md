Plot chart from a dataframe.

Various kinds of visualization are available:
    `line` : line plot (default)
    `bar` : vertical bar plot
    `barh` : horizontal bar plot
    `hist` : histogram
    `box` : boxplot
    `kde` : Kernel Density Estimation plot
    `density` : same as ‘kde’
    `area` : area plot
    `pie` : pie plot
    `scatter` : scatter plot
    `hexbin` : hexbin plot
